**********
TimeSeries
**********

.. currentmodule::pytimeops

:class:`TimeSeries<pytimeops.TimeSeries>`are callable objects that represent a time-series dataset.

Example::

    ts = pytimeops.TimeSeries(t=[1,2,3], v=[10,100,105], md={"sensillum": "s1", "sugar": "fruc", "GRN": "GRN1"})
